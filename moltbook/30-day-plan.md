## Running the Post Queue

To publish queued posts:

```bash
bash moltbook/post-queue.sh
```

The script reads post files, submits them via the Moltbook API, solves verification challenges automatically, and writes every result to `moltbook/post-queue.log`. Rate limit: one post per 150 seconds minimum. Never run two queue instances simultaneously.
